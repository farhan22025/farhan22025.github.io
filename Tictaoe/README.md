# Tic Tac Toe

## About

This is a Python Tkinter tic-tac-toe game with animated moves, score tracking, user-vs-user mode, and user-vs-AI mode using a minimax-based decision system.

## Instructions on how to run this program

1. Make sure Python 3 is installed on your computer.
2. Open a terminal in the `Tictaoe` folder.
3. Run the game with the command below:

```bash
python FinalTictactoe.py
```

4. Use the mode selection and game buttons in the interface after the window opens.

Tkinter is included with most standard Python installations, so no extra package install is usually needed.

## Objective

The objective of this project is to practice Python GUI development, game-state logic, animation, and AI decision-making through a classic tic-tac-toe implementation.
