# Recipe Share

## About

This is a simple HTML-based recipe sharing project where users can enter recipe details such as the dish name, image link, ingredients, cooking instructions, and a YouTube video link.

## Instructions on how to run this program

1. Open the `Recipe_Share` folder.
2. Open `RecipeShare.html` in a web browser.
3. Fill in the form fields and submit a recipe to see it appear on the page.

This is a front-end only project, so recipes are displayed in the browser session and are not stored permanently.

## Objective

The objective of this project is to practice HTML form design, DOM manipulation, and simple interactive content rendering in the browser.
