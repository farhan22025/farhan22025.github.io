# Farhan's Portfolio Prototype V1

## About

This folder contains an earlier personal portfolio prototype built with HTML, CSS, JavaScript, and local images. It represents an early version of Farhan Alam's web portfolio design before the current live portfolio.

## Instructions on how to run this program

1. Keep the `images` folder in the same directory as the HTML, CSS, and JavaScript files.
2. Open `index.html` in any modern web browser.
3. If needed, use a simple local server to preview the project with the same folder structure intact.

## Objective

The objective of this prototype is to practice portfolio design, content presentation, image-based layout work, and front-end structuring for a personal website.
