# Deepfake Detection

## About

This is Farhan Alam's final-year research-focused project on deepfake detection, combining software engineering discipline with machine learning and experimentation.

## Instructions on how to run this program

1. This folder currently contains project documentation only.
2. Review the research notes and thesis-related materials stored in the project workspace.
3. When source code or experiment notebooks are added, create the required Python environment and run the experiments from this folder.

## Objective

The objective of this project is to study manipulated media detection, explore model behavior, and build a stronger research foundation in machine learning and applied data-driven software systems.
