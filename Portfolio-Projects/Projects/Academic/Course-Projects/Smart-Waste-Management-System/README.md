# Smart Waste Management System

## About

This is a system design project focused on planning a smart waste management platform through diagrams, structured documentation, and data-flow thinking.

## Instructions on how to run this program

1. This folder currently contains documentation rather than a runnable application.
2. Review the project files and design artifacts in this folder when they are added, such as ERD diagrams, UML diagrams, and data flow documents.
3. Use the documentation as the reference point for future implementation work.

## Objective

The objective of this project is to design a clear, scalable waste management solution by organizing entities, workflows, and system interactions before implementation.
